The Reduced Instruction Set Computer or RISC as we know it, is a type of microprocessor architecture that utilizes a small, highly optimized set of instructions. In this paper the design of an 8-Bit Risc Processor is proposed. The design is implemented on a Nexys 4 DDR board, the purpose is to make the board act as a microprocessor and let it execute a specific given program and display the programâ€™s output on the seven-segment display on the board. The implemented Risc processor has four stages that the instruction goes through when executing a program which are, Fetch, Decode, Execute and Write back.

This study aims to enable effective COVID19 CT-scan lung image classification by combining deep features with conventional handcrafted features to classify whether the patient is diagnosed with COVID19 or Pneumonia or normal lungs. In particular, the deep features are extracted from a pre-trained convolutional neural network model, namely the VGG19 model, at six different extraction levels. The deep features extracted at each level are analyzed using a features selection algorithm to identify the deep feature combination that achieves the highest classification performance.
